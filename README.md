# curso-sass-webpack
Curso Sass utilizando Webpack

## Sequência de execução:

Instalação das dependências:
```
npm install
```
Execução do gerador do style.css inline
```
npm run start:watch
```
Execução do html de exemplo rodando na porta 3000
```
npm run start
```
Geração do arquivo style.min.css
```
npm run build
```
